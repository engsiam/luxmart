## 🚀 Overview

LuxeMart is a high-performance modern e-commerce frontend focused on delivering a premium shopping experience with clean UI systems, responsive layouts, smooth interactions, and scalable frontend architecture.

The project is designed with a strong emphasis on:

- ⚡ Performance optimization
- 🎨 Modern premium UI/UX
- 📱 Fully responsive layouts
- 🧩 Reusable component architecture
- 🛒 Real-world e-commerce workflows
- ✨ Smooth animations & interactions

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/engsiam/luxmart.git
```

Install dependencies:

```bash
pnpm install
```

Run development server:

```bash
pnpm dev
```
